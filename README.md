- Javascript

- ReactQuery

## <a name="features">🔋 Features</a>

- 🎃 Authentication & Authorization with JWT
- 📝 Create Post
- 🗑️ Delete Post
- ❤️ Like/Unlike Post
- 💬 Comment to a Post
- 👥 Follow/Unfollow Users
- 📱 Completely Responsive
- 🔁 Repost/Retweet
- 🔊 Notification
- 🏢 Account Management
